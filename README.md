1. Pomodoro Project
   A timer that helps with project management.
   For this project, we need a clean frontend that will display timers, sessions, and projects to the user.
   On the backend, we need to create an elaborate structure that can store:

- timers within sessions
- sessions within projects

The recommended stack:

- Frontend: Next.js
- Backend: Python and FastAPI
- WebSockets or polling for live timer sync across tabs/devices
- Basic analytics: total focus time per project/session

Goal:

- Learn how to structure a full-stack application
- Understand how frontend and backend communicate
- Practice managing nested data relationships (projects → sessions timers)
